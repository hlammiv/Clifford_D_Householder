Rewrites terms of \doxylink{classringZ9_aa5b506e62cbb99f21074500288726117}{element} into a canonical basis. 

This ring has a $\mathbb{Z}$-\/basis given by $\zeta_9^i$ for $i=0,1,\ldots,5$. Thus, any has a unique representation using only the first six entries of \doxylink{classringZ9_aa5b506e62cbb99f21074500288726117}{ring\+Z9\+::element} with the last three entries being zero. The function does this by taking advantage of the relations  \[
    1 + \zeta_9^3 + \zeta_9^6 = 0
\] \[
    \zeta_9 + \zeta_9^4 + \zeta_9^7 = 0
\] \[
    \zeta_9^2 + \zeta_9^5 + \zeta_9^8 = 0
\]

\begin{DoxyWarning}{Warning}
Most of the member functions in this class assume that a given ring element is in its reduced form. The constructors which have a possibility to initialize the last three entries will call \doxylink{group__ringZ9Mutators_gae934b2a35b47e2b47df7deb00f937d85}{reduce()} automatically. Any member function (such as \doxylink{group__ringZ9NumberTheoretic_ga7473bb216dd367a71470702034989b64}{Galois\+Aut}) or operation (such as \texorpdfstring{$\ast$}{*}) are likely to produce non-\/reduced instances and must call \doxylink{group__ringZ9Mutators_gae934b2a35b47e2b47df7deb00f937d85}{reduce()} before terminating. 
\end{DoxyWarning}
