\doxysubsection{Detailed Description}
Instances of the class represent elements of the ring $ \mathbb{Z}[\zeta_9, \frac{1}{3}] $ where $ \zeta_9 $ is the primitive ninth root of unity $ e^{2\pi/9} $. 

This class allows for basic arithmetic with elements of $ \mathbb{Z}[\zeta_9, \frac{1}{3}] $ and provides member functions that do some basic number-\/theoretic operations. Elements of this ring are comprised of a numerator which is a sum of ninth roots of unity and a denominator which is a nonnegative power of 3\+:  \[
   3^{-b}\cdot\sum_{i=0}^8 a_i \zeta_9^i
  \] for $ a_i \in \mathbb{Z} $ and $ b \in \mathbb{Z}^{\geq 0} $. In this class, the varaible $ b $ is stored in \doxylink{classringZ9chi_a8f87171738b56f720cebb1af800fca49}{exp}, and the $ a_i $\textquotesingle{}s are stored in \doxylink{classringZ9chi_a5710f33e51529092de99c168c8482d6b}{ring\+Z9chi\+::element}.

\begin{DoxyNote}{Note}
The use of \textquotesingle{}chi\textquotesingle{} in the class title is in reference to an alternative definition for this ring, where we replace the generator 1/3 with $\chi := (1 - \zeta_9)^{-1} $. This replacement gives the same ring.
\end{DoxyNote}
