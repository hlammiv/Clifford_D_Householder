\doxysubsection{Detailed Description}
Instances of the class represent elements of the ring $ \mathbb{Z}[\zeta_9] $ where $ \zeta_9 $ is the primitive ninth root of unity $ e^{2\pi/9} $. 

This class allows for basic arithmetic with elements of $ \mathbb{Z}[\zeta_9] $ and provides member functions that do some basic number-\/theoretic operations. The ring itself consists of all possible sums of ninth roots of unity of the form\+:  \[
 \sum_{i=0}^8 a_i \zeta_9^i
\] for $ a_i \in \mathbb{Z} $.
