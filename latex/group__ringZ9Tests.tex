{\footnotesize\ttfamily bool ring\+Z9\+::is\+Zero (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption}) const}



Determines whether a ring element is zero. 

We check by determing whether the ring element can be expressed as a sum of "{}minimal vanishing sums of roots of unity."{} For ninth roots of unity, these minimal vanishing sums are precisely  \[
    1 + \zeta_9^3 + \zeta_9^6 = 0
\] \[
    \zeta_9 + \zeta_9^4 + \zeta_9^7 = 0
\] \[
    \zeta_9^2 + \zeta_9^5 + \zeta_9^8 = 0
\]

\begin{DoxyNote}{Note}
When in reduced form, checking whether a ring element is zero simply amounts to checking that all entries of \doxylink{classringZ9_aa5b506e62cbb99f21074500288726117}{ring\+Z9\+::element} are zero, but in some cases, is\+Zero is called before a ring element passes through \doxylink{group__ringZ9Mutators_gae934b2a35b47e2b47df7deb00f937d85}{ring\+Z9\+::reduce}, so our algorithm is slightly more complicated. 
\end{DoxyNote}
