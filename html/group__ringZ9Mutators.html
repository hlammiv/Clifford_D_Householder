void ringZ9::reduce ( )

Rewrites terms of element into a canonical basis.

This ring has a \(\mathbb{Z}\)-basis given by \(\zeta_9^i\) for \(i=0,1,\ldots,5\). Thus, any has a unique representation using only the first six entries of ringZ9::element with the last three entries being zero. The function does this by taking advantage of the relations

\[ 1 + \zeta_9^3 + \zeta_9^6 = 0 \]

\[ \zeta_9 + \zeta_9^4 + \zeta_9^7 = 0 \]

\[ \zeta_9^2 + \zeta_9^5 + \zeta_9^8 = 0 \]

Warning
Most of the member functions in this class assume that a given ring element is in its reduced form. The constructors which have a possibility to initialize the last three entries will call reduce() automatically. Any member function (such as GaloisAut) or operation (such as *) are likely to produce non-reduced instances and must call reduce() before terminating.
