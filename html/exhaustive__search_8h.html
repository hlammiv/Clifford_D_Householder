void handleCtrlC ( int  sig)

This code allows us to exit gracefully when Ctrl+C is used during runtime.

Since we are dealing with large data structures, using Ctrl+C during runtime might result in memory leaks since the execution is abruptly halted. Now, when Ctrl+C is called, the program is allowed to terminate, thus allowing any destructors to be called for the vectors we've used.
