◆ fieldNorm()

int ringZ9::fieldNorm ( ) const

Computes the product of all Galois conjugates (which is guaranteed to be an integer).

See here for some info and general properties of the field norm.

Note
As a consequence of the Fundamental Theorem of Galois Theory, in certain fields (such as \( \mathbb{Q}(\zeta_9) \), a field element is rational iff it is fixed by every Galois automorphism, since Galois automorpihsms distribute over multiplication, any Galois automorphism will simply permute the factors of the field norm, meaning the field norm must be an integer, so converting to int is always valid.

int ringZ9::fieldTrace ( ) const

Computes the sum of all Galois conjugates (which is guaranteed to be an integer)

See here for some info and general properties of the field trace.

Note
The field trace is a 'trace' in the linear algebra sense. We can view \( \mathbb{Q}(\zeta_9) \) as a vector space over the rationals, and an element \( a \) of this field gives a linear transformation \( x \longmapsto ax \) for each \( x \in \mathbb{Q}(\zeta_9) \). The trace of this map is exactly the field trace here. The field trace is again guaranteed to be an integer.

◆ GaloisAut()

ringZ9 ringZ9::GaloisAut ( int  k) const

Computes Galois conjugate of given instance.

The Galois group of \( \mathbb{Q}(\zeta_9) \) is the cyclic group of order six. The Galois group consists of field automorphisms from \( \mathbb{Q}(\zeta_9) \). By 'automorphism,' we mean abort bijective function form the field to itself which a) distributes over addition, multiplication, subtraction, and division, and b) maps 0 and 1 to themselves. Because of these properties, any Galois automorphism is defined uniquely by where it maps \( \zeta_9 \) (which must also get mapped to a root of unity). In particular, the Galois automorphisms in this case exponentiate the root of unity in each term to some power \( k \), where gcd(k,9) = 1. That is, each term \( a_i \zeta_9^i \) becomes \( a_i \zeta_9^{i*k} \).

Parameters
kThis integer (if relatively prime to 9) defines a Galois automorphism, and is used to exponentiate the rou in each term.
Warning
The parameter k defines a true Galois automorphism iff gcd(k,9) = 1. The member function does NOT test this gcd condition.

◆ quad()

int ringZ9::quad ( ) const

Computes the positive-definite quadratic form in Eq. (13) of this paper.

If the coefficients of a ring element are given by \( a_i \) for \( i = 0,\ldots , 5 \), then the quadratic form is given by

\[ frac{1}{4}( 2a_0 - a_3)^2 + 3a_3^2 + (2a_1 - a_4)^2 + 3a_4^2 + (2a_2 - a_5)^2 + 3a_5^2 \]

\[ = \sum_{i=0}^5 a_i^2 - a_0a_3 - a_1a_4 - a_2a_5. \]

◆ sdeChi()

int ringZ9::sdeChi ( ) const

If \( \mbox{sde}_\chi \) of instance is less than 6, this computes it using Theorem 3.6 of the Kalra paper.

Theorem 3.6 in Kalra et. al. states that an element of this ring is divisible by \( \chi^k \) for \( k < 6 \) iff the augmentation (see ringZ9::signedWeight) of the \( (k-1)\)-th formal derivative divided by \((k-1)!\) is divisible by 3. Our formal derivatives are calculated using formalDerivative.

The reason I created this is because Theorem 4.3 in the Kalra paper says that the sdes in terms of \( \chi \) in any entries in a unit vector (and therefore, also a unitary matrix) will be the same. This means, when looking through candidates for the diagonal and x_3 in ESA, we can disregard any combination whose sde's disagree.

Note
With our usage, we can assume this member function will only be called on ring elements whose \( \mbox{sde}_\chi \) is less than 6 (since this is equivalent to the ring element being indivisible by 3). Thus, if this function is used on an element divisible by 3, this function returns 6, even if the true sde is higher.
Returns
the sde in terms of \( \chi \) if less than 6, and 6 otherwise.

◆ tauFieldNorm()

int ringZ9::tauFieldNorm ( ) const

Performs field norm with respect to \( \mathbb{Q}(\zeta_9 + \zeta_9^8) \).

The field \( \mathbb{Q}(\zeta_9 + \zeta_9^8) \) is the unique real subfield of \( \mathbb{Q}(\zeta_9) \) and it has three Galois automorphisms (each of which are restrictions of those from \( \mathbb{Q}(\zeta_9) \)). Specifically, we multiply GaloisAut(1)*GaloisAut(2)*GaloisAut(5).

Warning
This member function assumes that the ring element used is a REAL element. The function will return garbage if any nonreal element is passed.

Counts the number of roots of unity used to express a ring element.

The weight is calculated by taking the sum of absolute values of each entry in ringZ9::element.
