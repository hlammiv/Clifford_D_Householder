Detailed Description

Instances of the class represent elements of the ring \( \mathbb{Z}[\zeta_9, \frac{1}{3}] \) where \( \zeta_9 \) is the primitive ninth root of unity \( e^{2\pi/9} \).

This class allows for basic arithmetic with elements of \( \mathbb{Z}[\zeta_9, \frac{1}{3}] \) and provides member functions that do some basic number-theoretic operations. Elements of this ring are comprised of a numerator which is a sum of ninth roots of unity and a denominator which is a nonnegative power of 3:

\[ 3^{-b}\cdot\sum_{i=0}^8 a_i \zeta_9^i \]

for \( a_i \in \mathbb{Z} \) and \( b \in \mathbb{Z}^{\geq 0} \). In this class, the varaible \( b \) is stored in exp, and the \( a_i \)'s are stored in ringZ9chi::element.

Note
The use of 'chi' in the class title is in reference to an alternative definition for this ring, where we replace the generator 1/3 with \(\chi := (1 - \zeta_9)^{-1} \). This replacement gives the same ring.

int ringZ9chi::element[9]
private

Array for the coefficients on each of the nine rou terms in the numerator.

We may represent elements of our ring as a fraction where the numerator is an element of \( \mathbb{Z}[\zeta_9] \) which we encode with this array. As in ringZ9::element,
the \( i \)-th entry of the array stores the integer coefficient on the \( \zeta_9^i \) term.

◆ exp

int ringZ9chi::exp
private

This variable stores the power of three in the denominator.

We may represent elements of this ring as a fraction where the denominator is a nonnegative integer power of three, which is stored in this variable. I could have made the variable not unsigned but when combined with ringZ9chi::cancel(), testing for membership in \( \mathbb{Z}[\zeta_9] \) now amounts to testing whether this variable is zero.
